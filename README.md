# contur-bot

Тендерный агент для поиска и коммерческой оценки закупок в Контур.Закупках. Агент не только отбирает формально подходящие тендеры, но и оценивает вероятность допуска, победы, рентабельность, риски и следующий шаг менеджера.

## Направления поиска

- газоанализаторы, газосигнализаторы и системы контроля загазованности;
- сенсоры газоанализаторов, их диагностика, замена, калибровка и обслуживание;
- промышленная запорная и трубопроводная арматура;
- промышленные весы и компоненты весоизмерительных систем;
- датчики Dittmer и совместимые промышленные датчики.

География: Россия, Казахстан, Беларусь и другие страны СНГ.

## Состав проекта

- `AGENTS.md` — роль агента, поисковые критерии, оценка шансов и ежедневный регламент;
- `IMPLEMENTATION_PLAN.md` — план развития тендерного штаба и интеграции с Контур API;
- `dashboard/` — локальная панель менеджеров с воронкой, фильтрами и карточками тендеров;
- `dashboard/data/tenders.json` — данные для панели;
- `reports/tenders/` — сформированные Excel-отчеты;
- `CHANGELOG.md` — журнал выполненных работ.

## Запуск панели

Откройте `dashboard/index.html` напрямую или запустите локальный веб-сервер из корня проекта:

```powershell
python -m http.server 8765
```

После этого панель доступна по адресу `http://127.0.0.1:8765/dashboard/`.

Статусы менеджеров в текущем прототипе сохраняются локально в браузере. После получения доступа к Контур API планируется автоматическая загрузка закупок и централизованное хранение статусов.
